Reflexión del proyecto – Login App con React Native
PARTE 1 – Cómo usé la IA

Durante el desarrollo del proyecto usé inteligencia artificial principalmente para ayudarme a crear la pantalla de Login en React Native. Al principio le pedí ayuda para generar la estructura básica del proyecto y entender cómo usar componentes como View, Text, TextInput y TouchableOpacity. También le pedí ejemplos de código para hacer un formulario de inicio de sesión y mostrar mensajes de error o éxito según si el usuario ingresaba correctamente los datos.

Además, la IA me ayudó cuando tuve problemas técnicos con Expo Go y la conexión entre la computadora y el celular. Por ejemplo, tuve errores al escanear el QR, problemas con el modo Tunnel y mensajes como “Failed to parse manifest JSON”. Le fui mostrando los errores y me fue indicando paso a paso cómo solucionarlos hasta que la aplicación finalmente funcionó en el celular.

Otra cosa en la que me ayudó fue en el diseño visual. Le mostré la imagen de referencia que nos dieron en la consigna y le pedí un layout parecido. A partir de eso pude modificar colores, tamaños y posiciones de los componentes hasta lograr una interfaz similar a la del ejemplo.

PARTE 2 – Decisiones que tomé

Aunque la IA me generó gran parte del código base, tuve que hacer varias modificaciones manualmente. Por ejemplo, primero el login estaba configurado con un usuario distinto y después decidí cambiarlo por uno personalizado (andres) y mantener la contraseña 1234. También adapté algunos textos y detalles visuales para que quedaran más parecidos a la maqueta de la consigna.

Otra decisión importante fue revisar los errores en vez de copiar todo automáticamente. En un momento el login no funcionaba aunque los datos estaban bien escritos. Entonces tuve que revisar el código, recargar Expo y usar console.log para entender qué estaba pasando. Eso me ayudó a no depender solamente de la IA y a entender mejor cómo funciona la validación en React Native.

PARTE 3 – Qué aprendí

Con este proyecto aprendí a crear una aplicación básica en React Native usando Expo, a manejar estados con useState y a trabajar con componentes nativos para construir interfaces móviles. También aprendí a resolver errores relacionados con Expo Go y a entender mejor cómo funcionan los eventos y validaciones en JavaScript. Además, entendí que la IA puede ayudar mucho para programar y resolver problemas, pero que igualmente es importante revisar el código, probarlo y entender qué hace cada parte antes de usarlo.
